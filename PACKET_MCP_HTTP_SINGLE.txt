# Packet Grok Playability Gap Report

## Status

Identified and documented (March 2026).

This is not a bug report.
This is an architectural honesty document for the codebase.

## Why this document exists

The current direction of:

- `packetcli2machine`
- `packet_machine`
- `packet_mcp`

is clear:

> make it possible for another machine to observe and eventually play Packet for real,
> not only read static reports

The first serious external-machine test exposed a real gap:

- machine-readable is already here
- truly machine-playable from the outside is not here yet

## What already works

Right now an external machine can:

- read single-file bundles
- understand Packet machine render reports
- understand the MCP server structure
- simulate likely next states from code and reports
- propose commands and scenarios

That is useful.

But it is still not the same as touching the live runtime.

## The real gap

The current first MCP pass is local:

- stdio
- local process
- local PTY access

That means it works for:

- this machine
- local agentic environments with shell access

But not for a machine that only has:

- Python sandbox
- HTTP access
- no compiler
- no PTY
- no persistent local subprocess

## Why this matters

If we want:

- Grok
- Claude-like remote systems
- other external machines

to play Packet for real,
then the local stdio MCP pass is not enough.

It proves the interface.
It does not yet solve the outside-machine access problem.

## Phase 3 name

This suggests a new phase:

## Phase 3: True Machine Playability

Meaning:

- not only local machine observation
- but real external-machine access to live Packet runtime

## Minimal honest next step

The minimal next step is not public hosting.

The minimal next step is:

- a local HTTP pass over the same `PacketMachineBridge`

Suggested new file:

- `packet_mcp/server_http.py`

## Why HTTP pass first

Because it preserves the current architecture:

- `Packet` remains C runtime
- `packet_machine` remains Python bridge
- `packet_mcp` remains protocol layer

And only changes transport for outside clients.

## Expected HTTP shape

Minimal HTTP server should:

- run locally
- expose tool calls over JSON
- use the same `PacketMachineBridge`

Enough endpoints:

- `GET /health`
- `GET /tools`
- `POST /call`

## Example target flow

External machine sends:

```json
{
  "name": "packet_run_script",
  "arguments": {
    "commands": "v,SPACE,w,d,n,QUIT"
  }
}
```

And receives:

- real report text
- real latest snapshot
- real snapshot count

all produced by the actual live runtime

## What this would unlock

Then an external machine could:

- run real scripted Packet sessions
- see real crystal growth
- see real hashes
- see real turbulence emergence
- file bugs about actual runtime behavior, not only reports

## What this phase is not

Phase 3 is not yet:

- public internet hosting
- public GitHub deployment
- Docker-first distribution
- ProcessLang API

Those may come later.

Phase 3 is only:

- true external machine access to live Packet runtime

## Success condition

This document becomes obsolete when an outside machine can honestly say:

> I ran a real Packet command sequence against live Packet runtime
> and observed new state directly from the server

instead of:

> I inferred the next state from previous logs and code

```

# Packet Machine To MCP Hypothesis

## Status

First implementation pass.

## Why this should exist

`packet_machine` already acts like a machine-facing protocol layer:

- it launches real Packet runtime
- it sends scripted input
- it captures stable snapshots
- it writes a machine-readable report

This is already close to MCP thinking.

The missing piece is not world logic.
The missing piece is protocol formalization.

## Main claim

We should not start from "Packet itself becomes MCP".

We should start from:

- `Packet` stays low-level C runtime
- `packet_machine` becomes Python bridge
- `packet_mcp` becomes MCP server over that bridge

That gives us:

- real runtime below
- machine-friendly Python layer in the middle
- standard machine protocol on top

## Why not start with full MCP first

If we start directly from protocol,
we risk drowning in transport and schema work.

If we first expose a clean Python bridge API,
then MCP becomes only a wrapper over an already coherent machine interface.

## Minimal first MCP surface

First MCP server does not need the whole world.

It only needs a small honest surface:

- `packet_run_script`
- `packet_get_latest_report`
- `packet_get_latest_snapshot`

Optionally later:

- `packet_set_default_binary`
- `packet_set_default_commands`
- `packet_list_presets`
- `packet_run_preset`

## First tool semantics

### `packet_run_script`

Input:

- command string
- optional binary path
- optional timing parameters

Output:

- report text
- latest snapshot
- snapshot count

### `packet_get_latest_report`

Output:

- most recent machine render report text

### `packet_get_latest_snapshot`

Output:

- most recent snapshot only

## Why this is enough

With only these tools,
an external machine can already:

- cause Packet runtime to evolve
- observe results
- reason about behavior
- ask for another run

That is already much closer to "playing Packet"
than only reading docs and static reports.

## Transport

First pass should be:

- local
- stdio
- no internet server

This keeps the system simple and honest.

## Scope boundary

This first MCP pass is not:

- full remote hosting
- multiplayer
- full game API
- ProcessLang API

It is only:

- Packet runtime observation and scripted interaction via a standard machine protocol

## Success condition

This hypothesis is validated when:

- a local MCP-compatible client can connect
- it can list Packet tools
- it can call `packet_run_script`
- and it receives a real report generated from a real `packet_cli` run

## What this is

`packet_mcp` is a first local MCP server over `packet_machine`.

It is not a remote internet service.

- `packet_get_latest_snapshot`

From project root:

python3 packet_mcp/server.py

This is a first implementation pass.

It is intentionally small:

- local
- stdio
- no external dependencies
- built over real `packet_cli` through `packet_machine`

```python
from __future__ import annotations

from dataclasses import dataclass
from pathlib import Path

from .packet_machine_render import run_machine_render, write_report


@dataclass
class PacketMachineResult:
    binary: str
    commands: list[str]
    report_text: str
    latest_snapshot: str
    snapshot_count: int


class PacketMachineBridge:
    def __init__(self, default_binary: str = "./packet_cli") -> None:
        self.default_binary = default_binary
        self.latest_result: PacketMachineResult | None = None

    def run_script(
        self,
        commands: str,
        *,
        binary: str | None = None,
        startup_ms: int = 150,
        between_ms: int = 180,
        max_read_ms: int = 600,
        report_path: str | None = None,
    ) -> PacketMachineResult:
        binary_path = binary or self.default_binary
        command_list = [token.strip() for token in commands.split(",") if token.strip()]
        snapshots = run_machine_render(
            binary=binary_path,
            commands=command_list,
            startup_ms=startup_ms,
            between_ms=between_ms,
            max_read_ms=max_read_ms,
        )

        if report_path is None:
            report_path = "packet_machine_render_report.txt"

        out_path = Path(report_path)
        write_report(out_path, binary_path, command_list, snapshots)
        report_text = out_path.read_text(encoding="utf-8")
        latest_snapshot = snapshots[-1][1] if snapshots else ""

        result = PacketMachineResult(
            binary=binary_path,
            commands=command_list,
            report_text=report_text,
            latest_snapshot=latest_snapshot,
            snapshot_count=len(snapshots),
        )
        self.latest_result = result
        return result

    def get_latest_report(self) -> str:
        if self.latest_result is None:
            return ""
        return self.latest_result.report_text

    def get_latest_snapshot(self) -> str:
        if self.latest_result is None:
            return ""
        return self.latest_result.latest_snapshot

#!/usr/bin/env python3
from __future__ import annotations

import json
import sys
from pathlib import Path
from typing import Any


REPO_ROOT = Path(__file__).resolve().parent.parent
if str(REPO_ROOT) not in sys.path:
    sys.path.insert(0, str(REPO_ROOT))

from packet_machine.bridge import PacketMachineBridge


PROTOCOL_VERSION = "2024-11-05"
SERVER_NAME = "packet-mcp"
SERVER_VERSION = "0.1.0"


class PacketMcpServer:
    def __init__(self) -> None:
        self.bridge = PacketMachineBridge()

    def _send(self, payload: dict[str, Any]) -> None:
        body = json.dumps(payload, ensure_ascii=False).encode("utf-8")
        sys.stdout.write(f"Content-Length: {len(body)}\r\n\r\n")
        sys.stdout.flush()
        sys.stdout.buffer.write(body)
        sys.stdout.buffer.flush()

    def _ok(self, request_id: Any, result: dict[str, Any]) -> None:
        self._send({"jsonrpc": "2.0", "id": request_id, "result": result})

    def _err(self, request_id: Any, code: int, message: str) -> None:
        self._send(
            {
                "jsonrpc": "2.0",
                "id": request_id,
                "error": {"code": code, "message": message},
            }
        )

    def _read_message(self) -> dict[str, Any] | None:
        headers: dict[str, str] = {}
        while True:
            line = sys.stdin.buffer.readline()
            if not line:
                return None
            if line in (b"\r\n", b"\n"):
                break
            text = line.decode("utf-8").strip()
            if ":" in text:
                key, value = text.split(":", 1)
                headers[key.strip().lower()] = value.strip()

        length_text = headers.get("content-length")
        if not length_text:
            return None
        length = int(length_text)
        body = sys.stdin.buffer.read(length)
        if not body:
            return None
        return json.loads(body.decode("utf-8"))

    def _handle_initialize(self, request_id: Any) -> None:
        self._ok(
            request_id,
            {
                "protocolVersion": PROTOCOL_VERSION,
                "serverInfo": {"name": SERVER_NAME, "version": SERVER_VERSION},
                "capabilities": {"tools": {}},
            },
        )

    def _handle_tools_list(self, request_id: Any) -> None:
        self._ok(
            request_id,
            {
                "tools": [
                    {
                        "name": "packet_run_script",
                        "description": "Run packet_cli through packet_machine and return a real Packet machine render report.",
                        "inputSchema": {
                            "type": "object",
                            "properties": {
                                "commands": {"type": "string"},
                                "binary": {"type": "string"},
                                "startup_ms": {"type": "integer"},
                                "between_ms": {"type": "integer"},
                                "max_read_ms": {"type": "integer"},
                            },
                            "required": ["commands"],
                        },
                    },
                    {
                        "name": "packet_get_latest_report",
                        "description": "Return the latest Packet machine render report text.",
                        "inputSchema": {"type": "object", "properties": {}},
                    },
                    {
                        "name": "packet_get_latest_snapshot",
                        "description": "Return the latest Packet snapshot only.",
                        "inputSchema": {"type": "object", "properties": {}},
                    },
                ]
            },
        )

    def _tool_text(self, text: str) -> dict[str, Any]:
        return {"content": [{"type": "text", "text": text}]}

    def _handle_tools_call(self, request_id: Any, params: dict[str, Any]) -> None:
        name = params.get("name")
        arguments = params.get("arguments", {})

        if name == "packet_run_script":
            commands = str(arguments.get("commands", "")).strip()
            if not commands:
                self._err(request_id, -32602, "packet_run_script requires commands")
                return
            result = self.bridge.run_script(
                commands=commands,
                binary=arguments.get("binary"),
                startup_ms=int(arguments.get("startup_ms", 150)),
                between_ms=int(arguments.get("between_ms", 180)),
                max_read_ms=int(arguments.get("max_read_ms", 600)),
            )
            self._ok(
                request_id,
                self._tool_text(
                    "\n".join(
                        [
                            f"snapshot_count: {result.snapshot_count}",
                            "",
                            result.report_text,
                        ]
                    )
                ),
            )
            return

        if name == "packet_get_latest_report":
            self._ok(request_id, self._tool_text(self.bridge.get_latest_report()))
            return

        if name == "packet_get_latest_snapshot":
            self._ok(request_id, self._tool_text(self.bridge.get_latest_snapshot()))
            return

        self._err(request_id, -32601, f"Unknown tool: {name}")

    def handle(self, message: dict[str, Any]) -> None:
        method = message.get("method")
        request_id = message.get("id")
        params = message.get("params", {})

        if method == "initialize":
            self._handle_initialize(request_id)
            return
        if method == "tools/list":
            self._handle_tools_list(request_id)
            return
        if method == "tools/call":
            self._handle_tools_call(request_id, params)
            return
        if method == "notifications/initialized":
            return
        if method == "ping":
            self._ok(request_id, {})
            return

        if request_id is not None:
            self._err(request_id, -32601, f"Unsupported method: {method}")

    def serve(self) -> int:
        while True:
            message = self._read_message()
            if message is None:
                return 0
            self.handle(message)


def main() -> int:
    return PacketMcpServer().serve()


if __name__ == "__main__":
    raise SystemExit(main())

```python
#!/usr/bin/env python3
from __future__ import annotations

import argparse
import json
import sys
from http.server import BaseHTTPRequestHandler, ThreadingHTTPServer
from pathlib import Path
from typing import Any
from urllib.parse import urlparse


REPO_ROOT = Path(__file__).resolve().parent.parent
if str(REPO_ROOT) not in sys.path:
    sys.path.insert(0, str(REPO_ROOT))

from packet_machine.bridge import PacketMachineBridge


BRIDGE = PacketMachineBridge()


def json_bytes(payload: dict[str, Any]) -> bytes:
    return json.dumps(payload, ensure_ascii=False).encode("utf-8")


def tool_defs() -> list[dict[str, Any]]:
    return [
        {
            "name": "packet_run_script",
            "description": "Run packet_cli through packet_machine and return a real Packet machine render report.",
            "inputSchema": {
                "type": "object",
                "properties": {
                    "commands": {"type": "string"},
                    "binary": {"type": "string"},
                    "startup_ms": {"type": "integer"},
                    "between_ms": {"type": "integer"},
                    "max_read_ms": {"type": "integer"},
                },
                "required": ["commands"],
            },
        },
        {
            "name": "packet_get_latest_report",
            "description": "Return the latest Packet machine render report text.",
            "inputSchema": {"type": "object", "properties": {}},
        },
        {
            "name": "packet_get_latest_snapshot",
            "description": "Return the latest Packet snapshot only.",
            "inputSchema": {"type": "object", "properties": {}},
        },
    ]


def call_tool(name: str, arguments: dict[str, Any]) -> dict[str, Any]:
    if name == "packet_run_script":
        commands = str(arguments.get("commands", "")).strip()
        if not commands:
            return {"ok": False, "error": "packet_run_script requires commands"}
        result = BRIDGE.run_script(
            commands=commands,
            binary=arguments.get("binary"),
            startup_ms=int(arguments.get("startup_ms", 150)),
            between_ms=int(arguments.get("between_ms", 180)),
            max_read_ms=int(arguments.get("max_read_ms", 600)),
        )
        return {
            "ok": True,
            "name": name,
            "snapshot_count": result.snapshot_count,
            "latest_snapshot": result.latest_snapshot,
            "report_text": result.report_text,
        }

    if name == "packet_get_latest_report":
        return {"ok": True, "name": name, "report_text": BRIDGE.get_latest_report()}

    if name == "packet_get_latest_snapshot":
        return {
            "ok": True,
            "name": name,
            "latest_snapshot": BRIDGE.get_latest_snapshot(),
        }

    return {"ok": False, "error": f"unknown tool: {name}"}


class PacketHttpHandler(BaseHTTPRequestHandler):
    server_version = "packet-mcp-http/0.1.0"

    def log_message(self, format: str, *args: Any) -> None:
        return

    def _send_json(self, status: int, payload: dict[str, Any]) -> None:
        body = json_bytes(payload)
        self.send_response(status)
        self.send_header("Content-Type", "application/json; charset=utf-8")
        self.send_header("Content-Length", str(len(body)))
        self.end_headers()
        self.wfile.write(body)

    def do_GET(self) -> None:
        path = urlparse(self.path).path
        if path == "/health":
            self._send_json(200, {"ok": True, "service": "packet-mcp-http"})
            return
        if path == "/tools":
            self._send_json(200, {"ok": True, "tools": tool_defs()})
            return
        self._send_json(404, {"ok": False, "error": "not found"})

    def do_POST(self) -> None:
        path = urlparse(self.path).path
        if path != "/call":
            self._send_json(404, {"ok": False, "error": "not found"})
            return

        length = int(self.headers.get("Content-Length", "0") or "0")
        raw = self.rfile.read(length)
        try:
            payload = json.loads(raw.decode("utf-8")) if raw else {}
        except json.JSONDecodeError:
            self._send_json(400, {"ok": False, "error": "invalid json"})
            return

        name = payload.get("name")
        arguments = payload.get("arguments", {})
        if not isinstance(arguments, dict):
            self._send_json(400, {"ok": False, "error": "arguments must be object"})
            return

        result = call_tool(str(name), arguments)
        self._send_json(200 if result.get("ok") else 400, result)


def main() -> int:
    parser = argparse.ArgumentParser(
        description="Local HTTP pass for Packet MCP over PacketMachineBridge."
    )
    parser.add_argument("--host", default="127.0.0.1")
    parser.add_argument("--port", type=int, default=8123)
    args = parser.parse_args()

    httpd = ThreadingHTTPServer((args.host, args.port), PacketHttpHandler)
    print(f"packet_mcp_http listening on http://{args.host}:{args.port}")
    try:
        httpd.serve_forever()
    except KeyboardInterrupt:
        pass
    finally:
        httpd.server_close()
    return 0


if __name__ == "__main__":
    raise SystemExit(main())
